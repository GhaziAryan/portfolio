Capital One

Next-gen assessment platform

The challenge

Risk assessment workflows were scattered across tools, with no shared language or structure. Tight timelines compounded the problem — there was no room for misalignment between engineering, design, and methodology.

My approach

I centered everything around stakeholder relationships and human-centric design. Time-blocking for deep discovery. Speaking to users relentlessly. Building personal rapport to unlock alignment. Then ruthlessly prioritizing to ship the Scoping module and diagnostics recommender on schedule.

Key decisions

  • Led Camunda BPMN modeling to standardize workflows
  • Built Question Bank MVP as a questionnaire-as-a-service
  • Evangelized to 100+ stakeholders to validate demand
  • Defined testable success criteria before writing a line of code

What I learned

Enterprise platform adoption is a people problem before it's a product problem. The best roadmap is useless if you haven't built the relationships to execute it. I also mentored three associate PMs through this — teaching is the fastest way to sharpen your own thinking.

The challenge

Getting genuine value from a local LLM. The input needed to feel completely free-form — inviting real commentary and emotional reflection — while the model extracted structured, actionable behavioral insights behind the scenes. All without a server round-trip.

My approach

Fine-tuned prompts and built guardrails around a free-text input field that feels conversational but produces structured output. Used Claude Code, Cursor, and ChatGPT not just as copilots but as full workflow accelerators — PM discovery, code scaffolding, and iteration loops.

The AI-native thesis

Tape is a proving ground. It's less about the trading journal and more about testing what it means to build products when AI removes the barrier between idea and prototype. It's the future of PM — and I'm living in it.

Stack & tools

  • Swift + on-device LLM inference
  • Claude Code for rapid development
  • Cursor for code generation
  • ChatGPT for discovery & prompt engineering
